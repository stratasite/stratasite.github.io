## AI in the editor

Type a sentence in the filter drop zone instead of hunting for a field, and ask
for AI resolution to the right field or filter. AI is sprinkled throughout the
view editor to improve usability.

## An agent API

The same interactive interface that validates a partial query per keystroke —
compatible next steps, structured rejections — is structurally an agent API.
That's why Strata suits AI agents as well as humans, with grain-safe retrieval
throughout.
